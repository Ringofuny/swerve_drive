## conversion

- データを送信できる形にする関数（半分に分ける）
  ex) [ 1001 0010 1111 1100 ] の時 [ 1001 0010 ](上位バイト)と[ 1111 1100 ](下位バイト)に分けて
  それぞれ上位バイトは[ High_Byte ] に 下位バイトは[ Low_Byte ] に代入する関数

- データをCANで送って制御できる形にして返す（int16_t）
  -10(A) ~ 10(A) を -16384 ~ 16384 の間のデータに変換する関数
